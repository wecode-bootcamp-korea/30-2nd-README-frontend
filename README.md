## 구현 사항
1. 카카오 소셜 로그인 (회원가입 / 로그인)
2. 메인페이지 - 슬라이드, 카테고리별 리스트 필터링
3. 도서 상세페이지 - 더보기(페이지네이션), 리뷰
4. 장바구니 - 체크박스

## 구현 파트
1. 김준영 : Login/SignUp
2. 이희수 : Nav/Footer/Main
3. 한영현 : Detail/Cart
